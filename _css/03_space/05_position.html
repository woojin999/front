Display 속성을 좀 더 유동적으로 사용할 수 있게 위치값을 부여하는 것

  • Position: static => default, 별도 지정하지 않음
  • Position: fixed => 화면(screen영역)특정 위치에 고정
  • Position: sticky => 스크롤에 반응하여 특정 위치에 고정
  • Position: relative => 엘리먼트들끼리 위치값에 영향을 줌(관계가 있다)
  • Position: absolute => 엘리먼트의 위치를 절대적인 위치로 배정 (기준이 되는 엘리먼트는 relative 속성을 갖는 가장 가까운 조상 엘리먼트) 만약에 가장 가까운 조상 태그가 relative속성을 갖고 있지 않으면? document body (body 태그로 생각해도 무방함)
