상위 폴더로 돌아가기

heading tag는 왜 사용하는 것일까?
글자크기와 굵게 처리만 해서 대체사용하면 되는데?
- block 속성을 갖으므로 단독적인 강조 텍스트를 표현하기 좋다
- 컨텐츠들의 영역을 구분하는 기준점으로도 사용된다

h1 태그는 메인 주제 => 상위 문서로 돌아가기

부주제, 소주제 : h1을 주제로 사용하고 있다면 가급적 h3를 쓰자 => 크기 차이

부주제, 소주제

1. html 태그는 head태그와 body태그로 이루어져있다

2. head 태그는 문서의 형식, 추가정보, 스타일, 스크립트 등의 외부 리소스 정보를 담는다

3. body 태그는 웹 문서의 실질적인 컨텐츠(텍스트, 이미지 등)을 표현하는 영역이다

4. 문서의 출력에 관한 내용은 스타일링이 아닌 컨텐츠 및 기본 배치를 우선으로 고려한다
etc. 일반 텍스트 출력 비교
