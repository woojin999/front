HTML Tag를 element로 가져오는 메서드 byId, byClassName, byTagName

  • document.getElementById('idName') : 리턴 단일 엘리먼츠
  • document.getElementsByClassName('className') : 리턴 복수 엘리먼츠 : HTMLCollection 타입으로 저장
  • document.getElementsByTagName('idName') : 리턴 복수 엘리먼츠 : HTMLCollection 타입으로 저장
  • getElements로 가져오는 객체는 우선적으로 HTMLCollection의 index를 선택해야함을 기억해야한다
