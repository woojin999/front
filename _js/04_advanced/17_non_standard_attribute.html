html 태그의 속성이 제한적이므로 데이터관리 차원에서 개발자들에게 직접 속성을 만들 수 있는 방법이 제공됨

  • Markup 언어들은 사용자(개발자)들에게 표준이 아닌 사용자만의 문법을 사용할 수 있도록 설계되어 있음
  • HTML에서도 사용할 수 있는데 과도하게 비표준을 사용하면 개발효율과 가독성이 낮아질수 있음
  • 적당한 규칙을 적용한 비표준을 사용할 것을 권장함
  • 태그 속성의 이름을 data- 로 시작하여 하이픈(-) 뒤에 특정 의미를 갖는 이름을 사용할 수 있는데 이 규칙을 자바스크립트에서 인식할 수 있도록 설계되어 있음 : dataset property
  • dataset property를 사용할 때 대문자는 허용하지 않음 : 대문자를 사용하더라도 소문자로 변환시킴
