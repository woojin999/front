call, apply, bind 함수는 실행대상을 호출하여 실행을 위임하는 방식을 사용한다

  • targetObject.targetMethod.call(dataObject, parameter, parameter...) : 대상 객체의 내부 리소스(메서드)에 데이터 객체와 파라미터를 던져 수행하게 한다
  • targetObject.targetMethod.apply(dataObject,[ parameter, parameter...]) : 대상 객체의 내부 리소스(메서드)에 데이터 객체와 배열 던져 수행하게 한다
  • targetObject.targetMethod.bind(dataObject) : 대상 객체의 내부 리소스(메서드)를 가져와 데이터 객체에 숨겨놓고 가져온 리소스를 변수에 리턴한다. 추후 변수를 함수호출(variableName();)식으로 실행하면 데이터 객체의 메서드처럼 실행이 된다.
  • bind() 함수는 로드시점과 실행시점이 다르기 때문에 setTimeOut(callback, duration)과 같은 실행시차를 갖는 함수에서 실행할 때 this 키워드를 사용하더라도 데이터 객체를 그대로 참조할 수 있다.
