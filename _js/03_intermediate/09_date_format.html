날짜 및 시간 타입은 화면상으로는 문자처럼 보이지만 실제 날짜와 시간체계를 갖고 있는 데이터 타입이다

  • const now = new Date(); => 지금 이 순간의 날짜와 시간 값
  • const todayNow = new("2022-04-14T20:14:21Z"); => 오늘의 지금을 문자값으로 날짜 타입 만들기
  • 위의 T : 시간, Z:UTC 기준 시간
  • Local시간으로 조정하고 싶으면 Z대신 +HH:MM, -HH:MM 추가
  • 원리: Date()객체를 인스턴스화 하면 현재 순간의 날짜와 시간을 컴퓨터의 값에서 메모리로 호출하여 저장
  • 출력이 실생활에서 사용하는 형식이 아니므로 변환이 필수
  • Date.now() : 오래전부터 사용하는 오늘의 정보값을 리턴(ES5 부터사용)
  • 날짜와 시간은 정보의 처리에 관한 기록정보에 해당하므로 사용방법을 숙지하는 것을 추천
