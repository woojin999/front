break :특정 조건에서 반복을 중지하고 싶을 때 사용 continue : 특정 조건에서 실행을 건너뛰고 반복을 이어가고 싶을 때

  • 제어문에서 주로 사용되며 break; 키워드를 포함하고 있는 반복문을 중지하고 싶을 때 조건문 내부에 선언하는 것이 일반적
  • break; 가 작동되면 반복문 내에서 break; 아래에 있는 코드는 실행하지 않으므로 실행시점을 잘 팔단하여 사용해야 함
  • break;와 return을 동시에 사용할 때도 주의하여야 함
  • continue; 는 특정시점에서 조건을 수행 한 후 아래의 코드를 스킵하고 반복을 지속함
  • continue;는 반복문을 멈추는 것이 아니어서 잘못 이해하면 무한 반복을 만들어 낼 가능성이 있으니 주의해서 사용할 것
