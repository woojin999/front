자바스크립트의 배열은 유동배열 : 크기와 데이터가 변경이 가능하다

  • const myFavors = ['a', 'b', 'c', 1, 2, 3];
  • 원래 데이터 타입이 같아야 하지만 자바스크립트는 허용을 하고 있다
  • myFavors = []; 초기값이 없는 빈 배열을 만든 후 데이터를 추가하여 크기 변경이 가능하다
  • 각 데이터들은 콤마(,)로 분리되고 분리된 자리는 index로 관리되며 0부터 시작
  • 크기는 length(길이)라는 속성을 사용하여 측정하고 크기가 변경될 때마다 자동으로 산정된다
  • 인덱스 번호 체계에서 순서의 뛰어넘음 등의 변칙적인 사용은 모두 undefined로 초기화
  • 객체와 달리 index로 관리 되고 다양한 데이터를 저장할 수 있기 때문에 자바스크립트 프로그래밍에서 활용도가 매우 높으며 이해도가 낮으면 추후 데이터 통신 및 데이터 변환에서 프로그래밍이 힘들 수 있다
