데이터타입을 원하는 형식으로 변경하는 기법

  • string to number : Number("string number value"), parseInt("string integer value") , parseFloat("string float value")
  • number to string : 123.toString(), (123.45).toString()
  • string to boolean :Boolean("string value"), Boolean("")
  • boolean to string : String(true), String(false)
  • number to boolean : Boolean(number value), Boolean(1), Boolean(0)
  • boolean to number : Number(true), Number(false)
