변수나 상수에 할당 할 수 있는 값의 종류, 데이터의 구분 및 활용을 위해 매우 중요

  • 숫자 : Number : 정수(11, 100, 0 ,- 30), 실수(소수점이 있는 숫자), NaN(Not a Number : 숫자가 아님이라는 값) 문자 : string : '', "", ``으로 감싼 모든 값
  • 불리언 : boolean : 참/거짓 : true, false
  • 객체 : object : 1개 이상의 데이터가 특정 구조를 갖고 있는 형태 (아무것도 없는 사실도 1개로 취급)
  • 배열 : Array : [1, 2, ...] 형태로 이루어진 집합 데이터(순서, 길이 있음)
  • 미정된 값 : undefined : 값이 할당되지 않음
  • 알수없는 존재 : null : 무언가 있긴 하지만 파악이 안됨
  • Data type 알고 싶을 때 : typeof 변수 or 값
  1. 기본형 타입 (Primative Type): 값을 메모리의 주소값이 가르키는 곳에 직접 할당되어 있다
    number, boolean : string(원래 참조자료형이지만 사용의 편의성을 위해 기본 자료형처럼 사용할 수 있도록 변환됨)
  2. 참조 자료형 (Reference Type): 값이 저장되어 있는 메모리 주소값을 창조하고 있는 주소값이 할당되어 있다.
    나머지 string, object, array, funtion, null, undefined 등의 자료형
