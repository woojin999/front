이벤트 객체는 전역객체로써 모든 태그에 객체를 참조시킬 수 있다!

  • a 태그와 button 태그에 이벤트를 주로 참조시켜 클릭에 대한 반응을 기대할 수 있는 구조로 만드는게 일반적
  • a 태그는 href 속성에 의해 자체 이동(동기화 방식)이 구현되어서 별도의 이동속성을 막는 코드가 필요하다
  • 이벤트 객체를 부여하는 방식에는 크게 3가지가 있는데 각자 목적과 작동방식이 다르다

이벤트 연습을 위해 콘솔이 아닌 html문서(document영역)에 출력하는 방식을 이용할 예정

  • 화면출력영역을 document 영역(객체)이라고 한다
  • html 태그는 html Element, 속성은 attirbute, 일반 텍스트는 textNode라고 함
  • html로 출력 하기 : innerHTML 이라는 property를 사용
  • text로 출력 하기 : innerText라는 property를 사용
  • html element를 id값으로 객체 가져오기 : getElementById('idName')
  • html element를 css 선택자를 사용하여 가져오기 : querySelector('selector')
