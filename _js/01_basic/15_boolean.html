자바스트립트에서 boolean은 조건판단의 방법이 다양하여 혼란스러운 부분이 있다.

  • true :
    • 일반적인 데이터 타입 중 값이 있는 개념은 모두 true, 문자 띄어쓰기 포함
    • 정확히 정의 되지 않는 개념은 모두 false
    • Boolean("value")로 판단하여 조건판단을 유추해내는 방법이 있고, 조건항 혹은 조건문에 값을 대입하여 판단할 수도 있다
  • false :
    • 대표적으로 0, 빈문자열("", '', ``), undefined, null, NaN 이 해당함
    • 비교연산자, 논리연산자의 리턴은 모두 Boolean 타입
