class 속성은 주로 css를 위해 정의되므로 전용 메서드들이 있으면 편하다

  • addClass("classNameValue classNameValue ...") : 기존 클래스 속성값에 추가, 없으면 생성, 여러개 동시 생성 가능
  • removeClass("classNameValue classNameValue ...") : 일치하는 클래스값을 제거, 여러개 동시 제거 가능
  • toggleClass("classNameValue classNameValue ...") : 일치하는 클래스값이 있으면 제거, 없으면 생성, 여러개 toggle 가능
