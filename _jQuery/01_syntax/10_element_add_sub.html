태그 추가와 제거는 자바스크립트에서 동적구현의 핵심내용으로 기능 및 위치파악이 중요함

  • target.append('html sytax') : target의 자식으로 뒤쪽에 삽입
  • target.prepend('html syntax') : target의 자식으로 앞쪽에 삽입
  • target.after('html syntax') : target의 형제로 뒤쪽에 추가
  • target.before('html syntax') : target의 형제로 앞쪽에 추가
  • target.remove() : target을 삭제
  • target.remove('css selector...') : target중 css selector가 해당하는 것을 삭제 => 여러개 대상을 지정 가능
  • target.empty() : target 자식들을 삭제
