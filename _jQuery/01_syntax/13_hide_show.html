대상 엘리먼트를 숨기고 나타내는 효과는 자주 사용되므로 기억 해놓으면 편리하다

  • target.hide() : target 숨기기, target.hide(milliseconds) : milliseconds 동안 사라짐을 진행, target.hide(milliseconds, callback function) : callback function을 호출할수 있다
  • target.show(), target.show(milliseconds), target.show(milliseconds, callback function) :hide()메서드와 반대로 나타나게 함
  • target.toggle(), target.toggle(milliseconds), target.toggle(milliseconds, callback function) : target이 있으면 사라지게 하고 없으면 나타나게 하는 메서드
  • target.fadeIn(), target.fadeIn('keyword value'), target.fadeIn(milliseconds) : 천천히 나타나게 하는 효과 메서드
  • target.fadeOut(), target.fadeOut('keyword value'), target.fadeOut(milliseconds) : 천천히 사라지게 하는 효과 메서드
  • target.fadeToggle(), target.fadeToggle('keyword value'), target.fadeToggle(milliseconds) : toggle을 천천히 진행하는 메서드
  • keyword value : 'slow', 'fast'
  • fade 메서드는 진행속도 값 뒤에 최종 목표 투명도(0 ~ 1)값도 넣을 수 있다 : EX)target.fadeIn('slow', 0.3)
  • slideDown(), slideUp(),slodeToggle() : 슬라이딩 효과를 부여하는 메서드
  • 슬라이드 관련 메서드도 속도와 콜백함수 파라미터를 갖을 수 있다

    • Effect 메서드들은 jQuery slim 버전에는 미포함
