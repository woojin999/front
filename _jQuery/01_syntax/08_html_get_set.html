HTML 요소 가져오기(GET), 넣기(SET)

  • html(),text(), val() => innerHTML, innerText, value
  • html('html syntax'), text('string'), val('value') => innerHTML = 'html sytax', innerText= 'string', value = 'value'
  • attr('attribute name', 'value') => setAttribute('attribute name', 'value')
  • 다중입력시 파라미터를 객체에 선언하여 사용 : $("button").click(function(){ $("#w3s").attr({ "href" : "https://www.w3schools.com/jquery/", "title" : "W3Schools jQuery Tutorial" }); });
  • attr('attribute name') => getAttribute('attribute name');
  • css('property name', 'value') => style.propertyName = 'value', style.setProperty('property name', 'value');
  • css('property name') => style.propertyName, style.getPropertyValue('property name')
  • 다중입력 : css({"propertyname":"value","propertyname":"value",...});
